# Rust OpenAI Proxy Example

This example demonstrates how to trace LLM calls in a Rust agent by routing requests through the Solas Trace proxy server.

## Running the Example
1. Navigate to this directory:
   ```bash
   cd examples/rust/01_openai_proxy
   ```
2. Run the cargo binary:
   ```bash
   cargo run
   ```
3. Open `http://localhost:3000` to inspect the generated trace.

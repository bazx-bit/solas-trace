# Vercel AI SDK Integration Example

This example demonstrates how to trace text completions in a Vercel AI SDK application by routing operations through the Solas Trace proxy server.

## Setup & Running
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the example:
   ```bash
   npm start
   ```
3. Inspect `http://localhost:3000` to verify trace ingestion.

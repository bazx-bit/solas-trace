# TypeScript OpenAI Example

This example demonstrates how to integrate the official `openai` Node/TypeScript SDK with the Solas Trace Proxy.

## Setup & Running
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the example:
   ```bash
   npm start
   ```
3. Locate the corresponding `trace-ts-*` run in the Solas Trace dashboard to verify execution stats and prompts.
